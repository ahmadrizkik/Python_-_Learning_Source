
===== Catatan 1 =====
Cara menginstall sqlite
> Menginstall sqlite di windows dengan mendownload sqlite
yang bagian 'Precompiled Binaries for Windows' di web sqlite.
> Pindah ke sqlite di path C:\ buat folder nama sqlite
> view advanced system, open environment variables
> add new path in system variables c:\sqlite

===== Catatan 2 =====
DATABASE di sqlite
> database di sqlite terdiri dari table yang berisi rows dan column 
> akses melalui cmd dengan mengetik C:\Users\Rizky>sqlite3 newdb1.db
lalu akses .databases. Jika file tidak ada, akan membuat file baru
dengan nama newdb1.db
> buat tabel = CREATE TABLE user(rowid, name VARCHAR(25), email VARCHAR(35));
> VARCHAR adalah tipe data, perlu diinput saat awal
> akses .tables untuk melihat table user
> insert into untuk memasukkan data ke table
> INSERT INTO user(rowid, name, email) VALUES(1, 'alpha', 'alpha@skdj.osdf');
> INSERT INTO user(rowid, name, email) VALUES(2, 'beta', 'beta@opwq.ladf')
> update untuk mengubah data yang ada
> UPDATE user SET name='omega' WHERE rowid = 1;
> delete untuk menghapus data yang ada
> DELETE FROM user WHERE rowid = 2;
> print data dengan SELECT * FROM user;
